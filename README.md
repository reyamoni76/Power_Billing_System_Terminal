# Electricity Billing System

A Java console-based application for managing electricity bills.

## Features
1. User Registration and Login
2. Electricity Bill Calculation
3. Domestic and Commercial Connection Support
4. Bill Payment System
5. PDF Bill Generation
6. File Handling
7. Exception Handling

## Technologies Used
1. Java
2. Object-Oriented Programming (OOP)
3. File Handling
4. Exception Handling

## How to Run
1. Open the project in IntelliJ IDEA.
2. Run the Main class.
3. Follow the menu instructions displayed in the console.
